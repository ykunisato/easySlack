You can install the released version of easySlack from [CRAN](https://CRAN.R-project.org) with:

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(easySlack)
set_slack("GitHub access token", "#r", "Mr.R")
send_slack()
```
